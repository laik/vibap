## vibap

Install packages
```bash
npm install
# or
yarn install
```

Run the development server:
```bash
npm run dev
# or
yarn dev
```

Run the build server:
```bash
npm run build
npm run start
# or
yarn build
yarn start
```

Build and run docker image
```bash
make build && make run
```
